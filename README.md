#Overview

This project features a physical security container protected by a robust Two-Factor Authentication (2FA) system. It integrates embedded hardware with cloud services to ensure secure access control and real-time monitoring.

#My Role

I was responsible for developing the core programming logic using PlatformIO and C++. My work focused on managing the hardware-to-cloud communication, handling the authentication flow, and ensuring the seamless integration of various hardware modules on the ESP32-WROVER-E microcontroller.
